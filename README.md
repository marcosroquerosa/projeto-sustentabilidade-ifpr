# Projeto Sustentabilidade IFPR

Repositório oficial do projeto de pesquisa sobre sustentabilidade no Instituto Federal do Paraná (IFPR).

## 📌 Objetivo

Este repositório tem como finalidade centralizar e organizar os artefatos computacionais do projeto, incluindo:

- Notebooks de análise de dados
- Scripts e códigos utilizados no processamento e tratamento de dados
- Integrações e visualizações de dashboards
- Documentação técnica de apoio

O foco é garantir **transparência, reprodutibilidade e organização** das análises realizadas ao longo da pesquisa.



## 📂 Estrutura do Repositório

A organização do repositório segue a seguinte estrutura:
- projeto-sustentabilidade-ifpr/
- data/ # Dados utilizados (brutos e tratados)
- notebooks/ # Notebooks Jupyter para análise exploratória
- src/ # Scripts e códigos Python
- dashboards/ # Integração e documentação dos dashboards
- docs/ # Documentação adicional
- README.md # Descrição do projeto
